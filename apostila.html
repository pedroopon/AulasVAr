Comandos GitHub

git clone = selecionar ou criar um repositorio no GitHub (ir no repositorio no github e pegar o codigo https do repositorio)
git checkout -b feature/inicio = criar uma nova branch para não mexer na principal
git branch = mostrar as branchs disponiveis git add . = pegar os arquivos modificados
Git commit -m "descrição" = salvar as alterações
git push (git push -u origin feature/NovaPagina) = publicar no GitHub
git status = ver a situação dos arquivos
pull request = colocar as alterações e versões no arquivo oficial
no terminal usar o git bash
git checkout ... = trocar de branch
git pull = atualizar as informações dentro do VScode
